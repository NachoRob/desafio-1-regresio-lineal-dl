# Desafío – Introducción al Machine Learning

Este repositorio contiene el desarrollo del **Desafío: Introducción al Machine Learning** correspondiente a Desafío Latam, enfocado en la comprensión conceptual y la implementación práctica de un modelo de **regresión lineal** para la predicción del precio de viviendas.

## Información del estudiante
- **Nombre:** Ignacio Robles  
- **Grupo:** G-115 – Ciencia de Datos

## Descripción del proyecto

El desafío se divide en dos partes principales:

### Parte I – Preguntas conceptuales
Se abordan preguntas teóricas relacionadas con:
- Tipos de aprendizaje en Machine Learning.
- Diferencias entre enfoques supervisados y no supervisados.
- Fundamentos del modelamiento supervisado.
- Aplicaciones prácticas del Machine Learning en distintas industrias.

### Parte II – Implementación del modelo
Se desarrolla un flujo completo de modelamiento que incluye:
- Carga y exploración del dataset `data_housing.csv`.
- Análisis de calidad de datos y tratamiento de outliers.
- Codificación de variables categóricas.
- Entrenamiento de un modelo de regresión lineal.
- Evaluación del modelo mediante métricas y análisis de residuos.
- Interpretación de coeficientes e importancia de variables.

## Estructura del repositorio

- `Desafio_Introduccion_ML_Ignacio_Robles.ipynb`: Notebook principal con el desarrollo completo del desafío.
- `README.md`: Documento descriptivo del proyecto.

## Consideraciones finales

El objetivo de este trabajo es reforzar los conceptos fundamentales de Machine Learning y demostrar su aplicación práctica mediante un enfoque ordenado, reproducible y correctamente documentado.
